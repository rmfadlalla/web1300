Java Variables

Variables are containers that hold data in a program. In Java, each variable must have:

  1. An access specifier (except in the main method )
  2. a data type
  3. a name

All of these are stated in the variable declaration. A variable can be assigned a value in the same statement as the declaration with "=", or it can be done later.

Naming Conventions

When naming a variable, there are some rules and best practices to follow. Variable names:

  • should start with a lowercase letter
  • should follow camelCase, where each new word starts with an uppercase letter
  • may contain uppercase letters, lowercase letters, numbers, and the underscore character(_)
  • cannot contain spaces or other special characters like "/" and "?".
  • cannot be reserved words like "double" or "void".

Example Variable Declarations


private Access specifier int bottles;
String brand = "Pepsi";
public double cost = 1.99;
private int numberOfItems = 19;

Constants

A constant is similar to a variable, but cannot be changed while the program is running. Constant have the "final" reserved word in their declarations. Constant names should be in ALL_CAPS with words seperated by the underscore character(_)

Example Constant Declarations

final double SALES_TAX = 0.07;
