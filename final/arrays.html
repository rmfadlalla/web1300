Arrays

Array are used to hold multiple values of the same data type in one variable. Use square brackets after the data type to declare an array.



intData type of array[]This tells Java that this is an array valuesArray name = newThe "new" keyword tells the program to create a new array intData type[10This is the number of objects the array can hold. It cannot be changed later];
intData type of array[]This tells Java that this is an array numbersArray name = {1,3,5,7,9,11,13}Values can be added here instead of later by using curly brackets "{}";

You can modifiy the value of an element in an array using it's index number. Note that index numbers start from 0, so the first element's index is [0] the second's is [1] and so on

Two-Dimensional Arrays

A two dimentional array is an array of arrays. Think of it like a spreadsheet with rows and columns


int[][]The second set of square brackets tells Java this is a two-dimensional array awards = new int[10This is the number of rows][4This is the number of columns];

ArrayList

The ArrayList class allows you to create resizeable arrays. It has slightly different syntax from the built in arrays.

ArrayList<Circle> circles = new ArrayList<>;

Enhanced for Loop

The enhanced for loop is specificaly for looping through arrays. It cannot modifiy the contents of an array

for (int i:numbers) {
System.out.print(i + " ");
}
