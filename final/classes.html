About Java Classes

In Java, classes are like blueprints for creating objects. The class tells the computer what the objects contain and what you can do with them.

Parts of a Java Class

Every Java class is made up a few main parts:

  • A class declaration amd instance variables
  • A constructor
  • Mutators (also called Getters)
  • Accesors (also called Setters)

Each of these parts preforms it's own roll to make a class function.

Class Declaration & Instance Variables

This is where you put the name of the class and declare instance variables.

public class Circle {
private double radius;
}

Constructor

A constructor is used to create object of a class. It is also responseable for initializing instance variables.

public Circle() {
radius = 0;
}

Overloaded Constructor

An overloaded constructor can be used to set the value of instance variables for an object while it is being created

public Circle(double radius) {
this.radius = radius;
}

Mutators

Mutators are used to alter the instance variables of an object

public void setRadius(double radius) {
this.radius = radius;
}

Accesors

Accesors are used to retreive information from an object

public double getRadius() {
return radius;
}

Example Class

Circle Class

public class Circle {
private double radius;

public Circle() {
radius = 0;
}

public Circle(double radius) {
this.radius = radius;
}

public void setRadius(double radius) {
this.radius = radius;
}

public double getRadius() {
return radius;
}

public double getArea() {
return 3.14 * radius *radius;
}
}

CircleTester Class

public class CircleTester {
public static void main(String[] args) {
Circle circle = new Circle();
circle.setRadius(15);
System.out.println(circle.getRadius());
System.out.println(circle.getArea());
}
}
