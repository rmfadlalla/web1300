About Java Classes And Objects

In Java, classes are like blueprints for creating objects. The class tells the computer what the objects contain and what you can do with them.

Parts of a Java Class

Every Java class is made up a few main parts:

  • A class declaration amd instance variables
  • A constructor
  • Mutators (also called Getters)
  • Accesors (also called Setters)

Each of these parts preforms it's own roll to make a class function.

Class Declaration & Instance Variables

This is where you put the name of the class and declare instance variables.

A class name should start with an uppercase letter and be in CamelCase. It cannot contain spaces and should avoid using special characters. The class name should be the same as the name of the file that the class is in.

Variables in a class must have an access specifier (usualy private), a data type, and a name. See Variables

Methods

A method is a block of code that only runs when it is called. Methods make up most of a class, and each one has a specific purpose.

Constructor

A constructor is a specal method used to create objects of a class. It is responseable for initializing instance variables. It has no return type, not even void.

Overloaded Constructor

An overloaded constructor can be used to set the value of instance variables for an object while it is being created

/**
* Overloaded Constructor
* @param radius
*/
public Circle(double radius) {
this.radius = radius;
}

Mutators

Mutators are used to alter the instance variables of an object

/**
* Mutateor
* @param radius
*/
public void setRadius(double radius) {
this.radius = radius;
}

Accesors

Accesors are used to retreive information from an object

public double getRadius() {
return radius;

Example

Circle Class

public class Circle {
private double radius;

public Circle() {
radius = 0;
}

public Circle(double radius) {
this.radius = radius;
}

public void setRadius(double radius) {
this.radius = radius;
}

public double getRadius() {
return radius;
}

public double getArea() {
return 3.14 * radius *radius;
}
}

CircleTester Class

public class CircleTester {
public static void main(String[] args) {

Circle c1 = new Circle();
c1.setRadius(15);
System.out.println("c1: ");
