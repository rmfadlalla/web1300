Data Types

A data type tells Java what kind of information it is working with. Java data typs can be split into two groups, primitive and non-primitive

Primitive Data Types

Primitive data types have onle a size and type of value assigned to them and have no aditional methods. They start with lowercase letters and always take up the same amount of memory

Non-Primitive Data Types

These are called reference types these data types refer to Objects. They start with an uppercase letter.

Data Type Discription Example
int Holds an integer from -2,147,483,648 to 2,147,483,647 26
double holds a number with up to 15 digits and a decimal point 31.35
boolean Holds a true or false true
char Holds a single character 'J'
String Holds a sequence of characters "Hello, world! It's Auguest 14th!"
ArrayList Holds an array of objects see Arrays
